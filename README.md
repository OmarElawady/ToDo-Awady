# ToDo Backend Server
RESTful backend server for handling ToDo lists

## How To Use
In the directory of the API:
```
go run .
```

## Environment Variables
- DBFILE: location of the database.
- LISTENURL: URL backend server listen and serve on.

## Features
- Get all ToDo entries.
- Add new ToDo.
- Get ToDo entry.
- Update ToDo entry.
- Delete ToDo entry.
